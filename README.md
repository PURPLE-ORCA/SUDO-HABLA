# sudo-habla 🌮

Duolingo isn't going to help you when the production database is on fire. 

`sudo-habla` is a React-based terminal UI powered by a cynical, senior-dev AI. It forces you to learn practical Spanish by hooking directly into your workflow. Instead of teaching you how to say hello, it teaches you how to blame DevOps for a failed deployment. 

Run it globally, drop in an OpenAI-compatible API key (built for Kimi via Fireworks AI, but works with anything), and prepare to have your feelings hurt.

### The Commands
* **`/lore`**: Drops a cynical, battle-tested Spanish phrase of the day.
* **`/meaning <word>`**: Translates a tech concept with a real-world coding example.
* **`/roast`**: Grabs your latest `git diff` or commit message and brutally roasts your technical skills in Spanish, followed by the English translation and a tip on how to actually fix your garbage code.
